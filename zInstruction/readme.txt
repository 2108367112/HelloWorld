C++ 命名空间 namespace 的学习

//第一部分——类的声明   注意：类名首字母要大写。

class 类名{

    private:
    //1、构造函数(new 关键字是分配内存)
    //1、变量（需要赋初值的话，可以在这里就赋值，不能再类的实现中赋值。）
    //2、函数

    public:

    protect:

//注意：只有三种控制访问修饰符，如果不写的话，则默认为 private 。
}

//第二部分——类的实现——》在类的外部，实现类的成员函数。

返回值类型 类名::方法名{

    // :: 是作用域
}
